1.Enter line-bot file
	/d/line-bot
2.Run Python virtual environment (Name's venv)
	source venv/Scripts/activate
3.Run your program
	python app.py
4.Start ngrok server
	ngrok http 5000

Leave Python virtual environment
	deactivate
